# 🚀 CampusConnect – Enterprise Learning Platform (Next.js)

CampusConnect is a modern, responsive enterprise learning platform inspired by the Accredian Enterprise website. It is built using Next.js (App Router) with a focus on clean UI, reusable components, and smooth user experience.

## 📌 Project Overview

This project recreates a corporate learning platform landing page with:

- Fully responsive design (mobile + desktop)
- Smooth scrolling navigation
- Modular & reusable components
- Clean and structured UI
- Animated sections using Framer Motion
- Lead capture form with API integration

---

## 🛠️ Tech Stack

- Framework: Next.js 14 (App Router)
- Styling: Tailwind CSS
- Animations: Framer Motion
- Form Handling: React Hook Form
- Validation: Zod
- Icons: Lucide React
- Deployment: Vercel

## 🧠 Approach

1. Component-Based Architecture  
   Broke UI into reusable components (Navbar, Stats, Testimonials, etc.)

2. App Router Structure  
   Used Next.js App Router for scalability

3. Data Separation  
   Stored static content in a separate data file

4. Animation Strategy  
   Used Framer Motion for smooth UI interactions

5. Form Handling  
   Implemented lead capture form with validation and API route

## ⚙️ Setup Instructions

1. Clone the repository
git clone https://github.com/your-username/campusconnect-nextjs.git

2. Go to project folder
cd campusconnect-nextjs

3. Install dependencies
npm install

4. Run development server
npm run dev

5. Open in browser
http://localhost:3000
